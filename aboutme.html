Welcome to my portfolio website!

My name is Margaret Dorsey, and I'm a second year dual major Game Design and Development and Computational Math student at the Rochester Institute of Technology. I am also following the Computer Science curriculum above and beyond my minor requirements, and completing a second minor in the Japanese language. I also lead a club called Crashtest games, dedicated to enabling people to test each other's games and improve together, and I help every year to put on the Interactive Games and Media department megagame - designed and run by IGM students, faculty, and friends, lead by Professor Sean Boyle.

About this site

This site was originally developed for a web design and development class. For that project, I incorporated my own php to communicate with SMTP with gmail's SMTP server, and also used php extensively on many pages to simplify html and create some interesting single page applications. However, I hosted this version on heroku, which is unfortunately quite slow and cumbersome for how relatively light my portfolio is. As such, the site has been refactored to instead use javascript for functionality, so it can be hosted instead on git pages, which only supports static html. A link to the repository for this site through all its iterations can be found here.

Please keep in mind that this portfolio is still a work in progress - I plan to reincorporate the japanese language version and add more functionality as soon as I am able around my busy class schedule.
